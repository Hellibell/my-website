Thailand

แบบฝึกหัดทบทวน

1. ภูเขาไฟระเบิดมีสาเหตุจากข้อใด

  1. ชิ้นส่วนของภูเขาไฟได้รับความร้อนสูง จึงขยายตัวและเกิดการระเบิดอย่างรุนแรง
  2. แมกมา แก๊ส และไอน้ำที่ถูกอัดไว้เมื่อมีปริมาณเพิ่มขึ้นจะระเบิดออกมาทางปล่องภูเขาไฟ
  3. แมกมาเคลื่อนที่เข้ามาใกล้เปลือกโลก แก๊สจะแยกตัวออกจากแมกมาแล้วขยายตัวอย่างรวดเร็วจะเกิดการระเบิด
  4. แมกมาเคลื่อนตัวมายังบริเวณเปลือกโลก แล้วดันตัวออกทางช่องหรือรอยแตกของภูเขาไฟ ทำให้เกิดการระเบิด

2. สิ่งใดไม่พบบริเวณภูเขาไฟ

  1. พุแก๊ส (บ่อไอเดือด)
  2. หินบะซอลต์
  3. อัญมณี
  4. ซากดึกดำบรรพ์

3. ความหนืดของลาวาขึ้นอยู่กับสิ่งใด

  1. ปริมาณของแก๊สและไอน้ำ
  2. ปริมาณของแร่ซิลิกา
  3. ความรุนแรงในการปะทุของภูเขาไฟ
  4. รูปร่างของภูเขาไฟ

4. บริเวณใดที่เกิดแผ่นดินไหวที่รุนแรงและบ่อยครั้งที่สุดในโลก

  1. แนวเทือกเขาแอลป์ และเทือกเขาหิมาลัย
  2. แนวสันเขากลางมหาสมุทรอินเดีย และอาร์กติก
  3. แนวรอยต่อรอบมหาสมุทรแปซิฟิก
  4. แนวสันเขากลางมหาสมุทรแอตแลนติก

5. บริเวณใดที่มีโอกาสเกิดภูเขาไฟมากที่สุด

  1. แนวรอยต่อระหว่างแผ่นเปลือกโลก
  2. บริเวณที่มีการมุดตัวของแผ่นเปลือกโลก
  3. บริเวณที่มีการชนกันของแผ่นเปลือกโลก
  4. บริเวณที่มีการแยกห่างจากกันของแผ่นเปลือกโลก

6. บริเวณใดมีโอกาสเกิดคลื่นพื้นผิว

  1. ศูนย์เกิดแผ่นดินไหว
  2. รอบๆ ศูนย์เกิดแผ่นดินไหว
  3. รอบๆ จุดเหนือศูนย์เกิดแผ่นดินไหว
  4. ถูกต้องทุกข้อ

7. ความรุนแรงของภูเขาไฟระเบิดขึ้นอยู่กับข้อใด

  1. ความดัน
  2. ความร้อน
  3. พลังงานสะสม
  4. ความหนืดของแมกมา

8. ข้อใดถูกต้องเกี่ยวกับไซโมกราฟ

  1. ใช้บันทึกข้อมูลของคลื่นแผ่นดินไหว
  2. ใช้วัดความรุนแรงของภูเขาไฟระเบิด
  3. ใช้ตรวจวัดและหาทิศทางของคลื่นสึนามิ
  4. ถูกต้องทุกข้อ

9. การวัดขนาดของแผ่นดินไหวตามมาตราเมอร์คัลลีใช้หลักเกณฑ์ใด

  1. วัดความรุนแรงของแผ่นดินไหว
  2. ผลกระทบหรือความเสียหายที่เกิดขึ้น
  3. ความรู้สึกของคนที่ได้รับความเสียหาย
  4. วัดปริมาณพลังงานที่ปลดปล่อยออกมา

10. ไซสโมกราฟเป็นเครื่องมือที่ใช้ตรวจวัดสิ่งใด

  1. คลื่นสึนามิ
  2. แผ่นดินไหว
  3. ความดังเสียง
  4. ความเข้มแสง

11. ข้อใดไม่ใช่ลักษณะของคลื่นสึนามิ

  1. มีสมบัติเป็นคลื่นน้ำตื้น
  2. มีความยาวคลื่นมากกว่าคลื่นทะเลทั่วไปมาก
  3. มักเกิดจากแผ่นดินไหวบริเวณแผ่นเปลือกโลกมุดตัว
  4. เมื่อคลื่นเดินทางมายังบริเวณน้ำตื้นจะมีความเร็วมากขึ้น

12. ข้อใดกล่าวไม่ถูกต้องเกี่ยวกับคลื่นยักษ์สึนามิ

  1. คลื่นยักษ์สึนามิเกิดจากแผ่นดินไหวใหญ่ในทะเล
  2. เป็นการเคลื่อนที่ของเปลือกโลกตามแนวดิ่งและการแทนที่น้ำ
  3. การทดลองระเบิดนิวเคลียร์ใต้ทะเลไม่สามารถทำให้เกิดคลื่นยักษ์สึนามิได้
  4. การลดลงอย่างรวดเร็วของระดับน้ำทะเลตามทรายหาดเป็นสิ่งที่บ่งบอกว่าจะเกิดคลื่นยักษ์สึนามิได้

13. การเกิดแผ่นดินไหวแล้วส่งผลให้เกิดคลื่นยักษ์สึนามิในตอนใต้ของประเทศไทย เมื่อวันที่ 26 ธันวาคม พ.ศ. 2547 ศูนย์กลางของแผ่นดินไหวอยู่ที่บริเวณใด

  1. ทะเลจีนใต้
  2. มหาสมุทรอินเดีย
  3. มหาสมุทรแปซิฟิก
  4. มหาสมุทรแอตแลนติก

14. พลังงานที่ถูกปลดปล่อยออกมาจากการเกิดแผ่นดินไหว อยู่ในรูปของคลื่นชนิดใด

  1. คลื่นไหวสะเทือน
  2. คลื่นความร้อน
  3. คลื่นตามขวาง
  4. คลื่นแม่เหล็กไฟฟ้า

15. ข้อมูลใดผิดเกี่ยวกับมาตราเมอร์คัลลี

  1. เป็นการเก็บข้อมูลจากความรู้สึกของคนที่อยู่ในเหตุการณ์แผ่นดินไหวเท่านั้น
  2. ข้อมูลที่ได้สามารถแสดงออกมาเป็นแผนที่แสดงระดับความเสียหายได้
  3. ระดับความเสียหายจะลดลงเมื่ออยู่ห่างจากจุดเหนือศูนย์เกิดแผ่นดินไหวมากขึ้น
  4. การทำแผนที่แสดงระดับความเสียหายจากแผ่นดินไหวด้วยมาตราเมอร์คัลลีจะทำหลังจากเกิดแผ่นดินไหว

16. ข้อความใดกล่าวถูกต้อง

  1. คลื่นปฐมภูมิเป็นคลื่นตามยาว
  2. คลื่นทุติยภูมิจะมีความเร็วมากกว่าคลื่นปฐมภูมิ
  3. คลื่นปฐมภูมิจะเคลื่อนที่ผ่านได้เฉพาะตัวกลางที่เป็นของแข็งและของเหลวเท่านั้น
  4. อนุภาคของตัวกลางที่คลื่นปฐมภูมิเคลื่อนที่ผ่านจะเคลื่อนที่ในทิศทางตรงข้ามกับการเคลื่อนที่ของคลื่น

17. มาตราวัดความรุนแรงของแผ่นดินไหวที่สำรวจจากผลกระทบต่อมุนษย์และสิ่งปลูกสร้างคือมาตราใด

  1. มาตราริกเตอร์
  2. มาตราขนาดโมเมนต์
  3. มาตราเมอร์คัลลี
  4. มาตราโมส์

18. แอมพลิจูดของคลื่นไหวสะเทือนที่ตรวจวัดได้จากเครื่องตรวจวัดคลื่นไหวสะเทือน สามารถนำไปใช้ประโยชน์ในด้านใด

  1. คำนวณขนาดของแผ่นดินไหว
  2. คำนวณหาการเลื่อนตัวของรอยเลื่อน
  3. คำนวณหาความยาวของคลื่นแผ่นดินไหว
  4. คำนวณหาคาบอุบัติซ้ำของการเกิดแผ่นดินไหว

19. ข้อใดถูกต้องเกี่ยวกับเครื่องวัดความไหวสะเทือน

  1. กระดาษบันทึกความไหวสะเทือนจะบันทึกแรงสั่นสะเทือนในแนวดิ่งเท่านั้น
  2. กระดาษบันทึกความไหวสะเทือนจะบันทึกแรงสั่นสะเทือนในแนวราบเท่านั้น
  3. ลูกตุ้มเหล็กจะเคลื่อนที่ในทิศทางเดียวกับทิศทางการเคลื่อนที่ของคลื่นไหวสะเทือน
  4. ลูกตุ้มเหล็กจะเคลื่อนที่ในทิศทางตรงข้ามกับทิศทางการเคลื่อนที่ของคลื่นไหวสะเทือน

20. เหตุการณ์ใดเกิดขึ้นก่อนที่คลื่นสึนามิจะเคลื่อนที่เข้าสู่ชายฝั่ง

  1. น้ำทะเลลดลงอย่างรวดเร็ว
  2. ลมเปลี่ยนทิศทางอย่างฉับพลัน
  3. เกิดการโก่งตัวของแผ่นเปลือกโลก
  4. เกิดการสั่นสะเทือนของแผ่นดินไหว

21. การตรวจวัดเพื่อแจ้งเตือนการเกิดสึนามิสามารถทำได้โดยใช้เครื่องมือใด

  1. ดาวเทียมอุตุนิยมวิทยา
  2. เครื่องตรวจวัดคลื่นไหวสะเทือน
  3. ดาวเทียมสำรวจทรัพยากรทางทะเล
  4. ทุ่นตรวจวัดการเปลี่ยนแปลงระดับน้ำ

22. ข้อใดกล่าวถูกต้องเกี่ยวกับคลื่นสึนามิ

  1. คลื่นสึนามิจะเข้ากระทบชายฝั่งด้วยความเร็วที่เพิ่มขึ้นจากเดิม
  2. คลื่นสึนามิจะถูกยกตัวสูงขึ้นและรวดเร็วแต่จะลดต่ำลงเมื่อถึงชายฝั่ง
  3. คลื่นสึนามิที่เกิดในทะเลลึกจะมีขนาดใหญ่กว่าคลื่นสึนามิบริเวณชายฝั่ง
  4. การระเบิดอย่างรุนแรงของภูเขาไฟใต้ทะเลเป็นสาเหตุหนึ่งของการเกิดคลื่นสึนามิ

23. “การเกิดแผ่นดินไหวใต้ทะเลเป็นสาเหตุหลักเพียงอย่างเดียวที่ทำให้เกิดสึนามิ”

  1. ถูก
  2. ผิด

24. “หากอยู่บริเวณชายฝั่งแล้วเห็นระดับน้ำทะเลลดลงผิดปกติ และได้รับสัญญาณเตือนภัยสึนามิ ควรรีบออกจากพื้นที่ทันที”

  1. ถูก
  2. ผิด

25. “คลื่นสึนามิบริเวณกลางทะเลลึกจะมีความสูงมากและเคลื่อนที่เร็วมาก”

  1. ถูก
  2. ผิด

26. “แผ่นดินไหวปลดปล่อยพลังงานออกมาในรูปคลื่นไหวสะเทือนที่จุดเหนือศูนย์เกิดแผ่นดินไหวเท่านั้น”

  1. ถูก
  2. ผิด

27. “ไม่ควรสร้างบ้านเรือใกล้กับบริเวณที่เคยมีการปะทุของภูเขาไฟที่ดับสนิท”

  1. ถูก
  2. ผิด

28. “หากชาวประมงอยู่บนเรือหาปลางใกล้ชายฝั่ง และได้รับสัญญาณเตือนว่าเกิดสึนามิขึ้น ชาวประมงควรนำเรือออกไปกลางทะเลให้ไกลชายฝั่งมากที่สุด”

  1. ถูก
  2. ผิด

29. “ภูเขาไฟระเบิดเป็นสาเหตุทำให้เกิดแผ่นดินไหวได้”

  1. ถูก
  2. ผิด

30. “ศูนย์เกิดแผ่นดินไหว คือ ตำแหน่งที่เป็นจุดกำเนิดของแผ่นดินไหวซึ่งอยู่ภายในผิวโลก”

  1. ถูก
  2. ผิด
