Thailand

แบบฝึกหัดทบทวน

1. ภูเขาไฟระเบิดมีสาเหตุจากข้อใด

  1. ชิ้นส่วนของภูเขาไฟได้รับความร้อนสูง จึงขยายตัวและเกิดการระเบิดอย่างรุนแรง
  2. แมกมา แก๊ส และไอน้ำที่ถูกอัดไว้เมื่อมีปริมาณเพิ่มขึ้นจะระเบิดออกมาทางปล่องภูเขาไฟ
  3. แมกมาเคลื่อนที่เข้ามาใกล้เปลือกโลก แก๊สจะแยกตัวออกจากแมกมาแล้วขยายตัวอย่างรวดเร็วจะเกิดการระเบิด
  4. แมกมาเคลื่อนตัวมายังบริเวณเปลือกโลก แล้วดันตัวออกทางช่องหรือรอยแตกของภูเขาไฟ ทำให้เกิดการระเบิด

2. สิ่งใดไม่พบบริเวณภูเขาไฟ

  1. พุแก๊ส (บ่อไอเดือด)
  2. หินบะซอลต์
  3. อัญมณี
  4. ซากดึกดำบรรพ์

3. ความหนืดของลาวาขึ้นอยู่กับสิ่งใด

  1. ปริมาณของแก๊สและไอน้ำ
  2. ปริมาณของแร่ซิลิกา
  3. ความรุนแรงในการปะทุของภูเขาไฟ
  4. รูปร่างของภูเขาไฟ

4. บริเวณใดที่เกิดแผ่นดินไหวที่รุนแรงและบ่อยครั้งที่สุดในโลก

  1. แนวเทือกเขาแอลป์ และเทือกเขาหิมาลัย
  2. แนวสันเขากลางมหาสมุทรอินเดีย และอาร์กติก
  3. แนวรอยต่อรอบมหาสมุทรแปซิฟิก
  4. แนวสันเขากลางมหาสมุทรแอตแลนติก

5. บริเวณใดที่มีโอกาสเกิดภูเขาไฟมากที่สุด

  1. แนวรอยต่อระหว่างแผ่นเปลือกโลก
  2. บริเวณที่มีการมุดตัวของแผ่นเปลือกโลก
  3. บริเวณที่มีการชนกันของแผ่นเปลือกโลก
  4. บริเวณที่มีการแยกห่างจากกันของแผ่นเปลือกโลก

6. บริเวณใดมีโอกาสเกิดคลื่นพื้นผิว

  1. ศูนย์เกิดแผ่นดินไหว
  2. รอบๆ ศูนย์เกิดแผ่นดินไหว
  3. รอบๆ จุดเหนือศูนย์เกิดแผ่นดินไหว
  4. ถูกต้องทุกข้อ

7. ความรุนแรงของภูเขาไฟระเบิดขึ้นอยู่กับข้อใด

  1. ความดัน
  2. ความร้อน
  3. พลังงานสะสม
  4. ความหนืดของแมกมา

8. ข้อใดถูกต้องเกี่ยวกับไซโมกราฟ

  1. ใช้บันทึกข้อมูลของคลื่นแผ่นดินไหว
  2. ใช้วัดความรุนแรงของภูเขาไฟระเบิด
  3. ใช้ตรวจวัดและหาทิศทางของคลื่นสึนามิ
  4. ถูกต้องทุกข้อ

9. การวัดขนาดของแผ่นดินไหวตามมาตราเมอร์คัลลีใช้หลักเกณฑ์ใด

  1. วัดความรุนแรงของแผ่นดินไหว
  2. ผลกระทบหรือความเสียหายที่เกิดขึ้น
  3. ความรู้สึกของคนที่ได้รับความเสียหาย
  4. วัดปริมาณพลังงานที่ปลดปล่อยออกมา

10. ไซสโมกราฟเป็นเครื่องมือที่ใช้ตรวจวัดสิ่งใด

  1. คลื่นสึนามิ
  2. แผ่นดินไหว
  3. ความดังเสียง
  4. ความเข้มแสง
